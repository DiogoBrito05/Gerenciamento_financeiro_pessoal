# Ainda em desenvolvimento

## 📊 Sistema de Gerenciamento Financeiro Pessoal
Este é um projeto desenvolvido em Java com o objetivo de gerenciar finanças pessoais. Ele permite que o usuário registre transações, defina metas financeiras, acompanhe o fluxo de caixa e visualize relatórios financeiros detalhados. O projeto utiliza banco de dados para armazenar as informações de forma persistente.

## ⚙️ Funcionalidades
Cadastro de Transações: Registre receitas e despesas com categorias personalizadas, data, valor e descrição.
Classificação por Categoria: Categorize transações para acompanhar melhor onde o dinheiro está sendo gasto.
Definição de Metas Financeiras: Estabeleça objetivos financeiros e acompanhe o progresso.
Relatórios e Gráficos: Visualize seu fluxo de caixa por meio de gráficos e relatórios mensais ou semanais.
Controle de Dívidas: Gerencie suas dívidas e empréstimos, registrando valores e datas de pagamento.
Exportação de Dados: Exporte dados financeiros para arquivos CSV ou PDF para análise externa.
## 🛠️ Tecnologias Utilizadas
Java: Linguagem principal do projeto.
JDBC: Para comunicação com o banco de dados.
MySQL ou PostgreSQL: Banco de dados utilizado para persistência dos dados.
JavaFX ou Chart.js: Biblioteca de gráficos para representar visualmente as transações.
JasperReports (opcional): Para geração de relatórios em PDF.
## 🚀 Como Executar o Projeto
Pré-requisitos
Java 17+ instalado
Banco de dados MySQL ou PostgreSQL configurado
Dependências gerenciadas pelo Maven
